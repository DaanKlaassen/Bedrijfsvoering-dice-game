# Liar's Dice
## Dit project is bedoeld voor de opendag bij het gilde in Roermond.
### Dit is gemaakt met Godot en we willen laten zien hoe snel je programmeer concepten kan leren als je deze opleiding volgt.
# Algemene conventies
## Variable, Functies,Signalen: snake_case
### Constanten: ALL_CAPS
### Error handling: Gebruik assert waar mogelijk
### File and Directory benaming: camelCase
### Code Structure and Layout: 4 space indentaties
### 1 blank line per functie voor duidelijke spacing
